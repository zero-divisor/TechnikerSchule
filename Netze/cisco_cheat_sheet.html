IOS Modes

PromptWechseln mitErklärung
Switch> User Exec Mode / "view-only" Mode
Switch# enable Privileged Exec Mode
Switch(config)# configure terminal Global Config Mode
Switch(config-line)# line console 0 Line Configuration Mode
Switch(config-if)# interface FastEthernet 0/1 Interface Configuration Mode
exit ein Level zurück
end oder Ctrl+z zurück zu Privileged Exec Mode
disable von Privileged Exec Mode zu User Exec Mode

Basic Config

BefehlErklärung
Switch# show running-config Zeigt config im RAM
Switch# show startup-config Zeigt config auf dem NVRAM (Flash-Speicher)
Switch# write memory Speichert running-config dauerhaft
Switch# copy running-config startup-config Speichert running-config dauerhaft (gleich wie write memory)
Switch(config)# hostname SW1 Hostname zu SW1 ändern
Switch(config-line)# password test Setzt Passwort für User Exec Mode (line console 0)
Switch(config-line)# password test Setzt Passwort für alle vty lines (line vty 0 15)
Switch(config-line)# login aktiviert login prompt
Switch(config)# enable secret test Setzt Passwort für Privileged Exec Mode
Switch> show interfaces Gibt Infos zu allen Interfaces
Switch> show interfaces FastEthernet 0/7 Gibt Infos zu diesem Interface
Switch(config)# banner motd #test banner# Setzt message-of-the-day (MOTD) banner, delimiter ist vom Nutzer wählbar
Switch(config)# service password-encryption Verschlüsselt Passwörter in config (Schwache Verschlüsselung)
Switch# reload Verwirft running-config und lädt startup-config
Switch# erase startup-config Löscht startup-config, nach reload ist der Switch auf Werkseinstellungen zürückgesetzt

Configure Ip

BefehlErklärung
Switch# configure terminal Zu Global Config Mode wechseln
Switch(config)# interface vlan 1 Zu Interface Configuration Mode wechseln
Switch(config-if)# ip address 192.168.1.20 255.255.255.0 Ip Adresse und Subnetzmaske setzen
Switch(config-if)# no shutdown Interface (vlan 1) enablen
Switch(config-if)# exit Zurück zu Global Config Mode wechseln
Switch(config)# ip default-gateway 192.168.1.1 Gateway Ip setzen
